# Avatars Directory

This directory contains silhouette images for the onboarding flow.

## Required Images:
- `slim-silhouette.png` - Lean and athletic build
- `athletic-silhouette.png` - Muscular and toned build  
- `curvy-silhouette.png` - Fuller figure with curves
- `classic-silhouette.png` - Standard proportional build

## Image Specifications:
- Format: PNG
- Size: 200x300px recommended
- Background: Transparent or solid color
- Style: Simple silhouette outlines
